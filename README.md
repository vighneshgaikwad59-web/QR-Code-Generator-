# QR-Code-Generator-
QR Code Generator 🔗  A simple Python script that generates a QR code from text data using the qrcode library.
<br>
📋 Description

This project takes a string of data and encodes it into a scannable QR code image, saved locally as a PNG file. It's a beginner-friendly demo of working with the qrcode package in Python.

✨ Features
Generates a QR code from any text/string input
Saves output as a .png image
Minimal, easy-to-read code — great for beginner

Python 3
qrcode library
